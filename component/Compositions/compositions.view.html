Compositions

As the name alludes to, compositions are pre-made classes made to style commonly used elements. These compositions combine dashed utilities to speed up development.

Box

Box is a simple class that wraps the element in a borderless container. The said container has a drop shadow. While you'll typically add the color yourself, it will only color the drop shadow as they do lack a background color. 


// box - level - color
.box-low-primary
.box-med-primary
.box-low-primary

Buttons

Buttons Primarily comes in three types:

  • 1. Regular
  • 2. Fab
  • 3. Line

.btn {
  @extend .b-rounded, .font-md, .b-gray, .b-1-solid, .uppercase, .m-2,
    .cursor-pointer, .font-sans, .font-strong;
  // .btn-[color] is a flat button tha is colored.
  // btn - color
  .btn-primary
}

// btn - line - color
