Welcome to OrnateCSS!

May your code be clean, and your designs be ornate

Highly inspired by other frameworks like Tailwind css, and Bulma; Ornate CSS and built using Gaudiamus allows developers and designers to accomplish complicated web designs while you're writing your HTML. Whether you want to design using Ornate CSS' out-of-the-box solutions, or expand the utilities to fit your projects, Ornate empowers your decision while remaining light and non-invasive.

Basic Usage

Built directly into OrnateCSS is normalize.css, a css file that essentially resets a browsers default styling and forces uniformity for reliable consistency when styling your site. Normalize is definitely the recommended css reset of its kind. Regardless, use whatever you like as long as you have a reset of some sort.

Ornate has a plethora of utility classes. The majority of which follow this simple formula for ease of memorization: 


//name of property - side -unit number

// border - bottom - 1px - solid
.b-b-1-solid
  // margin - top - 1em
.m-t-1
  // margin - top & bottom - 1em
.m-y-1

As you see, when setting properties you can choose which side to affect. 


// Affect the entire border
// border - 1px - solid
.b-1-solid
  // margin - 3em
.m-3
  // Y axis e.g.: top and bottom
.m-y-2
  // X axis e.g.: left and right
.m-x-2

One can even review the above and notice a pattern when wit comes to declaring properties. Knowing that "m" stands for margin, you can then assume tha "p" stands for padding.

Pre-Built Utility Classes

There are two types of Utilities:
  • 1. Dashed Utilities
  • The most common and typically in the following format: .m-t-2
  • 2. Colon Utilities
  • Represent a condition that is met before being used. An example of this would be responsive utilities: "xs:", "sm:" , "md:", "lg:", "xl:", "xxl:", "xxxl:"
  • Meanwhile, the others represent a specific state. Such as focus, hover, and active.

In addition, there are default values that help support all the utilities. 


$baseSpacing 20px

$spacingUnit rem

$spacingStep .25

$numSpacingUnits 5

$shorthand-map: (
  "t":"top",
  "b":"bottom",
  "r":"right",
  "l":"left",
  "x":(1:"left", 2:"right"),
  "y":(1:"top", 2:"bottom"));

$property-map: ("m":"margin", "p":"padding");
